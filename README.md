# gip-projekt README

Extension that helps setting up VSCode projects for use in the university module GIP

## Features

Crate C++ projects with the right config.

## Requirements

OSX: `xcode-select install`
Windows: nothing
Linux: `apt-get install g++ gcc gdb`
